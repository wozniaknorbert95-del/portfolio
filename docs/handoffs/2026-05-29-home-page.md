## Key Decisions

- **No nested `<main>`**: Layout already wraps children in `<main>`, so page.tsx uses `<>` fragment
- **`ecosystem.json` import**: MetricsStrip imports JSON directly at build time via `resolveJsonModule`
- **Agent flow arrows**: SVG with `stroke-dasharray` + CSS `@keyframes` for animated dash movement; respects `prefers-reduced-motion`
- **Counter animation**: `useState` + `setInterval` for number count-up on scroll-into-view
- **All colors via CSS vars**: No hardcoded hex values except `#090c11` on primary CTA (dark text on blue accent)
